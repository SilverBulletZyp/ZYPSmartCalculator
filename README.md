# ZYPSmartCalculator

智能语音计算器


## 简介

以普通计算器科学输入方式为模板，基于讯飞语音SDK，可通过文字录入、算式录入及语音录入等方式计算结果，拥有语音识别、语音合成、语义理解等功能，是一款可以做到完全解放双手的智能语音计算器。
